## 核心内容

```yaml
 github.com/tealeg/xlsx/v3:
    Cell: # 数据单元格
        Value: # 设置Value值
        SetInt():
        SetValue():
    Col: # 数据列
    File: # 文件
        Sheet: # map映射
        Sheets:
        AddSheet(): # 添加数据表
        Save(): # 保存文件
    Row: # 数据行
        Cells:
        AddCell(): # 添加单元格
    Sheet: # 数据表
        Rows:
        AddRow(): # 添加数据行
    Style: # 单元格样式
    NewFile(): # 新建文件
    OpenBinary(): # 打开二进制文件
    OpenFile(): # 打开文件
```
