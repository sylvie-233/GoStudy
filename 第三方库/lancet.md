# lancet



## 基础介绍

一个功能强大的 Go 实用工具库

    /convert:

    /datetime:
        datetime: # 日期时间工具
            BeginOfDay():
            DiffInDays():
    /fileutil:
        fileutil: # 文件工具
            CopyFile():

        maputil: # Map 工具
            Keys():
