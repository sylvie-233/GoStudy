# goquery



## 基础介绍


Go 语言 HTML 解析库，提供了类似 jQuery 的语法来操作和查询 HTML 文档


## 核心内容
```yaml
github.com/PuerkitoBio/goquery:
    goquery:
        Document:   
            Find():
        Selection:
            Add():
            After():
            Append():
            AppendHtml():
            Attr():
            Before():
            Children():
            Each():
            Eq():
            Filter():
            FilterFunction():
            Find():
            Html():
            Last():
            Next():
            Parent():
            Prev():
            Text():
            ToggleClass():
        NewDocument():
        NewDocumentFromReader():
```
